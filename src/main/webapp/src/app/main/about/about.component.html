About

ChatterChatter as Discord alternative

The Application ChatterChatter is a chat application which enables people from all over the world to exchange with each other. You can meet anyone and communicate about anything. Just head over to the "Chat Rooms" page after logging in and create a new room or join an existing one.

Entities and Relations

There are three entities implemented:

  • users (read, create, update, delete)
  • chat rooms / subjects (read, create, delete)
  • messages (read, create, delete)

With two non-trivial relations:

  • user to message (1:n)
  • message to subject (n:1)

Validation

All input fields are validated in the frontend and the backend too.

Authentication

The creators of this website have implemented a working JWT-authentication to ensure maximum safety to all users.

Rich Client

The website ChatterChatter consists of two separate applications which are running simultaneously. The angular app provides the HTML, CSS and JavaScript while the backend, a spring boot application, takes care of the user management and persistence.

Responsive

ChatterChatter is a mobile friendly application and can be used on any device format.
